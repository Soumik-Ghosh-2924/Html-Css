Here we are going to learn about the various types of hovers, transitions and shadows effects that we can put as modifications in our buttons.
We can do this with the help of Pseudo class like "hover" and
and "active" and within these we can use css properties like
"opacity".

For implementing a smooth transition while hovering over the
buttons we have a CSS property that is known as "Transition"
which takes in two values:
the property and the time period.

we use "margin" property and "padding"
property to add proper spaces and orient
the elements properly in the web page.
Padding can be used instead of height
width and can be added on all the directions

By default the web browser aligns all the
elements based on the text position. Manually
also it can be done using the vertical-align
or using "align" in any other direction.

CSS specificity is an important property which tells
what the browser prioritises while displaying the web
page. It says that a class name selector has a more
priority than the element name priority.
