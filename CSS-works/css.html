Cascading Style Sheets

We use style tag.
The tags we are including in the style CSS are the elements which we are targetting
in the web page,
we have the CSS Selector, CSS Property and CSS Value.

"background color" is for the button's
background and the "color" property is
for the text color.

Similarly border and the height are the
property to modify the button in our own
way.px stand for the pixels.

rgb stands for red,green and blue value
for the color and for each the max value
is 255.Thus rgb{255,255,255} stands for the
white color.

border-radius property is for the rounded
corners and cursor is for the shape which
comes when the cursor is moved over to the
button.

for different button when we want to have
different styles we use the class atrribute
to label the button respectively and to implement
the different styles we use the syntax
".class-name" as the Target CSS Selector.

The margin-right property is used for adding space
to the right of and element and the value of so
is in pixels.
